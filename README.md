📌 PROJECT TITLE

Real-Time Chat Application Using Socket.IO

📖 PROJECT DESCRIPTION

This project is a Real-Time Chat Application developed using WebSocket technology through Socket.IO. The application enables multiple users to communicate instantly via a web interface with seamless frontend and backend integration.
The system maintains a live connection between clients and the server, ensuring instant message delivery without page refresh.

🏗 PROJECT ARCHITECTURE
Client (Browser)
       ↓
Socket.IO Connection
       ↓
Node.js + Express Server
       ↓
Broadcast to Other Clients

⚙ HOW THE APPLICATION WORKS

User opens the chat application in browser.
Browser establishes a WebSocket connection with the server.
When a user sends a message:
Message is sent to server via Socket.IO.
Server broadcasts message to all connected users.
All users see the message instantly.
